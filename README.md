# Challenge-number-1-Min-Tic
This is challenge one of the course of the ministry of technology of colombia in Phyton


Ciclo 1 Fundamentos de programación

Descripción: En la industria del desarrollo de software en muchas ocasiones se necesita
generar expresiones matemáticas y traspasar los datos de las variables, así pues se pide que
resuelva el siguiente problema
Requerimiento:

escriba una función con las siguientes condiciones
-Debe calcular la siguiente operación matemática:
a+bc
d+ef
y su resultado se almacene en una
variable llamada "x".
-La función también debe calcular la siguiente operación: a− b
c−d la cual debe ser
almacenada en una variable llamada "y".
-Una vez hecho lo anterior debe pasar el valor de "x" a la variable "y" y el valor de la
variable "y" a la variable "x".
-Solo puede realizar las operaciones matemáticas del primer punto, una vez.
-La función debe retornar el valor de "x" y "y"
Entradas Salida
a b c d e f Return
1 2 3 4 5 6 (3.0, 0.3448275862068965)
-1 2 -3 3 -5 6 (-0.6666666666666667, -0.7692307692307692)
